# Trae Switch 一键启动说明

## 🚀 使用方法（macOS）

### 方法 1：桌面启动器（推荐 - 完全无窗口）

你的桌面上已经有一个名为 **`TraeSwitch_启动器.app`** 的应用程序。

**使用方法：**
1. 双击 `TraeSwitch_启动器.app`
2. 系统会提示你输入管理员密码
3. 输入密码后，应用会在后台启动（**无任何窗口**）
4. 启动成功后会显示系统通知

**停止应用：**
- 目前没有一键停止的桌面快捷方式，需要用以下方法之一：
  - **方法 A**：打开终端，执行 `sudo pkill -f trae-switch`
  - **方法 B**：打开「活动监视器」，搜索 `trae-switch`，强制退出

### 方法 2：启动脚本（会显示终端窗口）

桌面上还有一个 **`启动.command`** 脚本。

**使用方法：**
1. 双击 `启动.command`
2. 输入管理员密码
3. 应用启动后可以关闭终端窗口

---

## 📁 文件说明

| 文件 | 位置 | 说明 |
|------|------|------|
| TraeSwitch 启动器 | 桌面（符号链接） | 一键启动，无任何窗口 |
| TraeSwitch_启动器.app | ~/trae-switch/ | 启动器应用本体 |
| 启动.command | 桌面 | 启动脚本 |
| 停止.command | ~/trae-switch/ | 停止脚本 |

---

## ⚠️ 注意事项

1. **必须使用管理员权限**：应用需要绑定 443 端口和修改 hosts
2. **启动器会提示输入密码**：这是正常的，因为需要 sudo 权限
3. **启动成功后无任何窗口**：应用会在后台运行，可通过系统通知确认
4. **关闭终端会影响应用**：如果你用终端启动，关闭终端会停止应用（启动器不受影响）

---

## 🔍 如何确认应用正在运行

### 方法 1：系统通知
启动成功后会显示「Trae Switch 已启动」的通知。

### 方法 2：终端命令
```bash
ps aux | grep trae-switch
```

### 方法 3：测试代理
```bash
curl -k https://127.0.0.1/v1/models
```
如果返回模型列表，说明代理正常。

---

## 🛑 如何停止应用

### 方法 1：终端命令（推荐）
```bash
sudo pkill -f trae-switch
```

### 方法 2：活动监视器
1. 打开「活动监视器」
2. 搜索 `trae-switch`
3. 选中进程，点击「强制退出」

### 方法 3：运行停止脚本
```bash
cd ~/trae-switch
./停止.command
```
